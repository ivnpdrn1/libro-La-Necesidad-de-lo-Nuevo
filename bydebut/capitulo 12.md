# **Capítulo 12 — El infierno lógico (con caso claro)**

La diferencia no fue inmediata.

El sistema siguió funcionando.

Ese fue el problema.

—Mira esto —dijo Thomas.

La pantalla mostraba un caso.

Nada extremo.

Una persona dentro del sistema.
Una decisión equivocada.

No era grave.

Pero tampoco era correcta.

—Antes esto se resolvía —dijo Susan.

Thomas asintió.

—Sí.

George añadió:

—Porque terminaba.

Thomas continuó.

—Antes, una persona tomaba una mala decisión. Eso generaba un problema. El sistema lo corregía con el tiempo. La persona perdía posición, perdía oportunidades, envejecía, salía del sistema.

Susan asintió.

—Se cerraba.

Thomas avanzó la pantalla.

—Ahora no.

El mismo caso.

La misma persona.

La misma decisión.

El problema aparecía.

Pero no desaparecía.

—Sigue ahí —dijo Susan.

—Sí —respondió Thomas.

George añadió:

—Y va a seguir.

La pantalla mostraba lo siguiente.

El sistema reaccionando.

Más controles.
Más supervisión.
Más reglas alrededor de la misma persona.

—Están corrigiendo —dijo Susan.

Thomas negó.

—No.

George la miró.

—Están rodeando el problema.

El caso continuaba.

La persona seguía tomando decisiones.

Algunas correctas.
Otras no.

El sistema seguía ajustándose.

Más capas.

Más procesos.

—Esto no termina —dijo Susan.

—No —respondió Thomas.

George añadió:

—Porque la persona no desaparece.

El silencio fue claro.

Antes, el tiempo sacaba a las personas del sistema.

Ahora no.

—Entonces el problema se queda —dijo Susan.

—Sí —respondió Thomas.

—Y el sistema crece alrededor —añadió George.

La pantalla mostraba el resultado.

El sistema no colapsaba.

Se volvía más pesado.

Más lento.

Más complejo.

—Esto no es solución —dijo Susan.

Thomas negó.

—Es mantenimiento.

George añadió:

—Y es continuo.

El caso avanzaba.

La misma persona.

Más decisiones.

Más ajustes del sistema.

Nada se eliminaba.

Todo se acumulaba.

—Esto no es un caso —dijo Susan.

Thomas respondió:

—Es el modelo.

George asintió.

—Se repite.

El silencio volvió.

Más claro.

Más inevitable.

—Entonces esto no tiene final —dijo Susan.

—No —respondió Thomas.

—Porque nadie sale —añadió George.

La frase quedó.

Porque explicaba todo.

—Antes el sistema se limpiaba —dijo Susan.

Thomas asintió.

—Sí.

George añadió:

—Ahora no.

Susan levantó la mirada.

—Entonces todo se queda.

Nadie la corrigió.

El caso seguía.

El sistema también.

—Esto no se rompe —dijo Susan.

—No —respondió Thomas.

George añadió:

—Sigue.

El silencio final no fue largo.

Pero fue suficiente.

Porque por primera vez, el problema no era el error.

Era que el error no terminaba.

Y que el sistema, en lugar de eliminarlo…

se estaba construyendo encima de él.
