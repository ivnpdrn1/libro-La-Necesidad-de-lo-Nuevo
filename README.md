# La necesidad de lo nuevo

Una historia sobre el límite, el amor y el peso de existir.

---

## 📖 Sobre el libro

Este proyecto explora qué ocurre cuando el límite deja de organizar el comportamiento humano.

No es una historia sobre la inmortalidad.

Es una historia sobre lo que sucede cuando el final deja de tener peso.

---

## 🧠 Concepto central

Cuando el final desaparece:

- el bien pierde urgencia  
- el mal pierde castigo  
- el amor pierde forma  
- el sistema pierde equilibrio
